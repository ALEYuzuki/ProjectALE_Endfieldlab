# Endfield Lab Starter (Next.js 15 + Tailwind)

- App Router, dark UI
- Simple i18n (ja/en) with URL prefix `/[locale]`
- Sidebar navigation matching the requested taxonomy
- Dashboard-like home

## Quick start

```bash
pnpm i
cp .env.local.example .env.local  # then fill secrets
pnpm dev
```

## Secrets
Never commit `.env.local`. Put your DeepL key and Sanity tokens there.
